字幕匹配 BDRip

魔法纪录 S2 第 4 集和第 7 集的音轨和画面不同步，我们推荐直接搭配 VCB 版 BDRip 观看，VCB 的 BDRip 的音轨已经过以下处理。本字幕的时间轴匹配的是处理后的视频。

+ 第 4 集音轨需要提前 0.2 秒（但是 OP/ED 不变）
+ 第 7 集音轨需要提前 0.15 秒 （但是 OP/ED 不变）

如果要搭配没有经过该处理的 BDRip 观看，请使用播放器的音轨同步功能，将音轨分别提前 0.2 秒 / 0.15 秒，在该情况下， OP/ED 会有错位，但是对观看影响不大。

请不要使用字幕同步功能提前延后显示字幕，因为 SweetSub 的字幕是以帧为单位对齐画面的，这么做会使得 （1） 画面字不同步 （2）字幕出现/消失时无法对齐场景切换的那一帧。 

这个音轨不同步现象在 WEB 源就已经有了，但是到 BD 也没有修。
